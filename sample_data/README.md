# Sample Data

This folder is where you can drop sample research paper PDFs for testing
PaperPilot AI locally.

No copyrighted sample PDF is bundled with this project. To get a free,
legally-redistributable test paper, download any open-access paper from:

- https://arxiv.org (e.g. search "attention is all you need")
- https://www.ncbi.nlm.nih.gov/pmc/ (open-access biomedical papers)
- https://openreview.net (many ML conference papers are open access)

Save the PDF into this folder, then upload it via the "Upload" page in the
running application (or via `POST /api/papers/upload`).
